AJAX (Asynchronous JavaScript and XML) adalah teknik yang memungkinkan pertukaran data antara server dan halaman web secara asinkron, tanpa harus memuat ulang seluruh halaman. Dengan AJAX, kita dapat memperbarui bagian-bagian tertentu dari halaman tanpa mengganggu pengalaman pengguna.

jQuery adalah pustaka JavaScript yang menyediakan metode dan fungsi bantu untuk memanipulasi elemen HTML dan berinteraksi dengan server. Ada beberapa kebelihan jQuery yaitu menggunakan selektor gaya CSS untuk memilih elemen HTML, menyediakan metode bawaan untuk manipulasi, animasi, dan komunikasi dengan server, mengurangi jumlah kode JavaScript yang harus ditulis.
Cara Menggunakan jQuery:
1. Unduh jQuery dari situs resmi dan sertakan di HTML kita.
2. Atau gunakan CDN (Content Delivery Network) untuk menghubungkan ke jQuery.
